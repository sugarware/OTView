# OTView
Outline-Text Viewer

階層テキストビューアです  
閲覧、検索、の快適さに拘りました  
編集機能が無いので、誤操作等で大事な文書ファイルを壊す心配がありません  
メニューのボタンはタップ、ダブルタップ、長押し、と機能も階層化されて覚えやすい  
  
* text, markdown
* 3つのペーン(階層、本文、検索結果)
* 選べる見出し定義方式
* 階層範囲でのAND検索
* 可変レイアウト
* 可変文字サイズ
* 選べる表の罫線表示
* PWA対応で、通常アプリっぽく使える
